

# Storbandet Fokus – Landing Page

A dark and elegant single-page website for a big band / jazz ensemble, designed to attract event organizers and get booked for gigs.

## Visual Style
- **Dark background** with gold/warm accent colors for a sophisticated, jazzy feel
- Elegant typography with serif headings and clean sans-serif body text
- Subtle animations on scroll for a polished experience

## Sections

### 1. Hero
- Full-width hero with a large headline (e.g. "Storbandet Fokus") and a short tagline
- A prominent "Book oss" (Book us) call-to-action button
- Atmospheric background (dark gradient or placeholder for a band photo)

### 2. About / Om oss
- Brief description of the band — who they are, what they play, their style
- Highlight the band's experience and versatility (events, festivals, private parties, etc.)

### 3. Media
- Embedded video placeholder (e.g. YouTube/Vimeo performance)
- Photo gallery section with placeholder images
- Optional: links to Spotify or SoundCloud

### 4. Events / Konserter
- Upcoming gigs listed with date, venue, and short description
- Clean card-based layout
- Hardcoded sample events (no backend needed)

### 5. Contact / Kontakt
- Contact form with fields: Name, Email, Event date, Message
- Form submits via a simple toast confirmation (no backend)
- Social media links and email address displayed

## Footer
- Band name, copyright, and social media icons

## Technical Notes
- Single-page with smooth scroll navigation
- Fully responsive (mobile-friendly)
- No backend needed — all content is static/hardcoded
